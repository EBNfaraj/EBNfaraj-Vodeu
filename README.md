# 🎬 برنامج استراحة الفيديوهات
**Video Rest - Video Management Platform**

منصة ويب مفتوحة المصدر لإدارة الفيديوهات ورفعها وتنظيمها في قوائم تشغيل وصفحات مخصصة، مبنية بـ Flask و SQLite.

---

## ✨ المميزات | Features

- 📤 **رفع الفيديوهات** — دعم صيغ MP4 و WebM
- 🗂️ **قوائم التشغيل** — تنظيم الفيديوهات في قوائم متعددة
- 📄 **صفحات مخصصة** — إنشاء أقسام وصفحات لتصنيف المحتوى
- 🔍 **البحث** — البحث بالعنوان أو التاقات
- 📊 **إحصائيات** — تتبع المشاهدات والتحميلات
- 🔐 **نظام مسؤولين** — إدارة المستخدمين وحماية المحتوى
- 📱 **تصميم متجاوب** — يعمل على جميع الأجهزة

---

## 🚀 تشغيل المشروع | Getting Started

### 1. متطلبات التشغيل
- Python 3.9+
- pip

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. تشغيل البرنامج
```bash
python app.py
```

افتح المتصفح على: `http://localhost:5100`
